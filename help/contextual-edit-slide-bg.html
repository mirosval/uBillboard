Slide - Background

Background Image URL
The main field. You should almost always have an image for your slide. You can select from all images in your Media Library. A preview will be displayed on the left hand side of the slide edit box.
Apply Automatic Resizing
Ises internal solution to resize images to appropriate dimensions automatically.
Background Color
Fallback background color for the slide, in case you have transparent images or videos in your slide.
Transparent Background
Related to the above, but since you can not use color picker to select a transparent color, we have this checkbox.
Retina Image
If the image you have supplied to the Background Image URL field is at least double in resolution than uBillboard, tick this option to render and use retina version of the image automatically on the devices that support it.
